# pay_serviced

클라이언트 <-> 서버 <-> 카카오페이서버

클라이언트는 React를 사용하여 요청하였으며, 서버는 Spring Boot를 사용하였다.

본 프로젝트는 카카오페이 결제 API를 이용하는 서버이며, 클라이언트 요청에 파라미터 값으로 제품명, 가격 기타 등등의 값을 넘겨주면 서버가 받아서 카카오페이서버와 통신합니다.




[환경구성 및 사용법]
java 17 버전을 사용 \n
intellij 툴 사용\n
빌드도구 : gradle \n

사용법 : 해당 프로젝트를 로드하여 gradle.bulid를 실행시키면 된다.
